PSL Data Visualization

Created by the UCSC LINQS Lab

A PSL Visualization that can be used for model understanding and debugging. The application is made up of multiple modules that convey information from different layers of a PSL model. These modules are organized by: model information, ground rule information, and ground atom information. We hope this tool can help PSL developers create the best models possible! The projects source code can be found on Github.

Provide a .json file from a PSL run with the -viz flag
